# Support

AIMeter is a community-maintained, unofficial utility.

## Questions and bugs

Open a GitHub issue for reproducible bugs, setup problems, provider page parsing changes, or feature requests.

Please include:

- macOS version
- Xcode version, if the issue is build-related
- AIMeter version or commit
- affected area: Cursor tracking, build, packaging, or documentation
- clear reproduction steps

## Privacy

Do not share live credentials, cookies, provider account IDs, billing details, or unredacted usage screenshots in public issues.

For security-sensitive problems, follow [SECURITY.md](SECURITY.md) instead of opening a public issue.
